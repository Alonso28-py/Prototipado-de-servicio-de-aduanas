# 🛂 Prototipo SGAI - Paso Pehuenche

Bienvenido al repositorio del **Sistema de Gestión Aduanera Integrada (SGAI)**. Este proyecto se enfoca en la iteración rápida de prototipos frontend para validar las reglas de negocio en la frontera (Acuerdo Chileno-Argentino).

## 🎯 Objetivo del Prototipo
Validar la interfaz de usuario (UI) y la experiencia de usuario (UX) del formulario de salida temporal de vehículos, antes de su integración con el backend.

## 🔄 Estrategia de Versionamiento e Iteraciones
El desarrollo se realiza mediante un enfoque iterativo:
*   **V1.0 (MVP Visual):** Estructura base del formulario con HTML y CSS (Tailwind). Sin validaciones lógicas.
*   *Próximas iteraciones:* Incorporación de validaciones en tiempo real (JavaScript) y generación de Código QR simulado.

## 🧪 Plan de Pruebas (ISO 25010)
A lo largo de las iteraciones, el prototipo será sometido a simulaciones de calidad bajo el estándar ISO 25010, enfocándonos en:
1.  **Adecuación Funcional:** Validar que el sistema distinga correctamente entre patentes normales (180 días) y diplomáticas (90 días).
2.  **Usabilidad:** Prevención de errores del usuario (campos vacíos o formatos incorrectos).
